# TAREA 1
```
INICIO
    suma = 0
    contador = 0

    LEER estatura

    MIENTRAS estatura ≠ 0 HACER
        suma ← suma + estatura
        contador ← contador + 1
        LEER estatura
    FIN MIENTRAS

    SI contador > 0 ENTONCES
        promedio = suma / contador
        ESCRIBIR "La estatura promedio es: ", promedio
    SI NO
        ESCRIBIR "No se ingresaron estaturas"
    FIN SI
FIN
```
